Employee Onboarding Request Automation – ServiceNow

📌 Project Overview

Developed an automated Employee Onboarding solution using the ServiceNow Service Catalog and Flow Designer to streamline new employee setup activities across departments.

🚀 Features

* Designed a Service Catalog item to capture employee and organizational details.
* Automated the onboarding process using Flow Designer.
* Auto-created Catalog Tasks (SCTASK) for the HR team.
* Configured assignment groups for proper task routing.
* Tested end-to-end workflow by generating RITM and SCTASK records.

🛠️ Technologies Used

* ServiceNow (Xanadu)
* Service Catalog
* Flow Designer
* ITSM Concepts

🔄 Workflow Explanation

1. User submits the Employee Onboarding Request from Service Portal.
2. ServiceNow generates Request (REQ) and Requested Item (RITM).
3. Flow Designer triggers on the requested item.
4. HR Catalog Task (SCTASK) is automatically created and assigned.

🎯 Outcome

Successfully automated the employee onboarding initiation process, reducing manual effort and improving task routing efficiency.
